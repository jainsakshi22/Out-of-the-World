Out-of-this-world
=================

Learning application from bitfountain.io

It is a sample project. This application contains some predefined planet on home(OWOuterSpaceTableViewController) controller. You can click on planetname to see the planet's image which will redirect to other controller(OWImageSpace viewController. You can either zoom or scroll the image. Also, clicking on planetname will redirect you to next (OWSpaceDataViewController) controller which will contain all the information about the planet. 
You can add new planet by clicking on add button and that planet information will be shown to home controller. You can delete the newly added planet by swiping on that planetname.
